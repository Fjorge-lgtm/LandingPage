HTML

O HTML é a linguagem base para a criação de páginas da web. Dominá-la é fundamental para qualquer pessoa
que deseja se tornar um desenvolvedor web. Você terá total controle sobre o visual e o funcionamento do seu site,
sem precisar depender de plataformas de criação online.

CSS

O CSS, ou Cascading Style Sheets, é uma linguagem de marcação que te permite estilizar e dar vida às páginas da web criadas com HTML.
Com o CSS, você pode controlar diversos aspectos visuais do seu site, como:cores, fontes, layout, fundos.
